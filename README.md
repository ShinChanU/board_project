## 1. Project Intro ❓

- SPA 게시판 프로젝트 입니다.
- 풀스택으로 웹 개발을 하고 싶었습니다.
- Node 애플리케이션으로 MERN STACK 을 채택했습니다.

## 2. Core-Tech 🛠

- <b>FrontEnd</b>
  Typescript, React
  Zustand, Styled-Component, Axios
  <br />
- <b>BackEnd</b>
  Express.js, MongoDB
